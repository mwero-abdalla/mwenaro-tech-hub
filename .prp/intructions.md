

### **Mwenaro Tech Academy — PRP **

**Product Name / Working Title:**
Mwenaro Tech Academy

**One-Sentence Idea Description:**
A responsive web platform for learners to take tech digital courses (self-paced or instructor-led), for instructors to manage learners and projects, and for admins to oversee courses, users, and payments — with AI-powered marking, recommendations, and certificate generation.

**Target Audience:**

* **Learners:** anyone looking to learn tech skills online
* **Instructors:** organization-assigned mentors managing cohorts
* **Admins:** managing courses, users, and site operations

**User Journey / Flow:**

* **Learners:**

  * Discover courses via homepage or catalog
  * Register with email/password or Google OAuth
  * Receive email with course/payment details
  * Access self-paced courses or join instructor-led sessions
  * Submit projects (hosted on GitHub/Google Docs)
  * Get AI-assisted grading with instructor final approval
  * Download certificate upon completion
  * Use dashboard to track progress, view assignments, and chat with instructors

* **Instructors:**

  * Login to dashboard
  * View assigned learners and cohorts
  * Review projects with AI suggestions
  * Manually grade if needed
  * Manage session links and schedules
  * Track learner progress
  * View monthly payments/payslips

* **Admin:**

  * Add/edit/remove courses and instructors
  * Manage learners and cohorts
  * View all dashboards and site-wide statistics
  * Oversee AI grading outcomes, session schedules, and payments
  * Manage chat and support

**Core Features:**

* Learner registration/login (email & Google)
* Course catalog with text/video materials
* Self-paced and instructor-led course modes
* AI-powered project marking, certificate generation, and course recommendations
* Dashboard for learners, instructors, and admins
* Quizzes, assignments, and progress tracking
* Chat system for learner-instructor feedback
* Support chat system
* Email notifications for sessions, payments, and course updates
* Payment integration for learners and monthly payslips for instructors
* Logs for user activity and analytics

**Suggested Tools / Stack:**

* **Frontend:** Next.js + React + Tailwind CSS + shadcn ui
* **Backend & Database:**  : Nextjs + Supabase (PostgreSQL)
* **Authentication:** Supabase Auth with email/password + Google OAuth
* **Payments:** Stripe or PayPal
* **AI & Automation:** OpenAI (recommendations, certificate generation, AI marking)
* **Project Hosting:** GitHub, Google Docs
* **Email:** SendGrid or Supabase Email

**Design Vibe & Color Scheme:**

* **Vibe:** Minimal, professional, visually appealing, and engaging — clean layout, clear typography, subtle design elements
* **Color Palette:**

 ColEnergetic coral (#E85D3B) with deep navy (#1E3054) - much more inviting!


**Future Feature Ideas:**

* Mobile app for learners and instructors
* AI tutors for personalized guidance
* Advanced analytics dashboards
* Gamification or course bundles
